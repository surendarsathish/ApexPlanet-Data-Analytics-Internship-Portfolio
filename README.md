# ApexPlanet Data Analytics Internship Portfolio

## Overview
This repository serves as the final capstone portfolio for my **ApexPlanet Data Analytics Internship**. It consolidates all the tasks completed during the internship into one professional master repository and demonstrates my understanding of the complete data analytics workflow — from **data cleaning and exploratory data analysis to dashboard development, data storytelling, and statistical validation**.

Throughout this internship, I worked on practical analytics tasks using tools such as **Python, Pandas, NumPy, Matplotlib, Seaborn, Excel, Power BI, Google Colab, and GitHub**. This portfolio represents my learning journey, project execution, technical growth, and ability to convert raw data into meaningful business insights.

---

## Internship Objective
The objective of this final portfolio is to integrate all completed internship tasks into one structured repository that reflects:
- My end-to-end understanding of data analytics
